# AI Trading Screener v3

## Soubory
- app.py              → hlavni aplikace
- requirements.txt    → zavislosti
- gitignore.txt       → prejmenuj na .gitignore
- streamlit_config.toml → uloz jako .streamlit/config.toml

## Spusteni lokalne
pip install -r requirements.txt
streamlit run app.py

## Nasazeni na Streamlit Cloud
1. Nahraj app.py a requirements.txt na GitHub
2. Vytvor slozku .streamlit/ a vloz config.toml
3. Nasad pres share.streamlit.io
